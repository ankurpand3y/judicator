## 1. Project Overview

**Judicator** audits LLM-as-a-Judge systems for bias. It does not evaluate models.
It evaluates the judges that evaluate the models.

### What it does
- Takes a user-defined judge: `Judge(llm_fn, system_prompt, eval_template)`
- Auto-detects judge type (pointwise / pairwise / binary)
- Shows cost estimate before running
- Runs 7 bias tests using pre-shipped fixtures
- Reports individual bias scores per test — no composite metric

### Public API (two imports only)
```python
from judicator import Judge, JudgeAuditor
```

Nothing else is top-level. `JudgeType` is internal.

## 2. Architecture Decisions (Locked)

1. **Two imports only.** `Judge` and `JudgeAuditor`. Nothing else in `__init__.py`.
2. **Library never handles API keys.** User wraps their own LLM callable.
3. **Library makes zero LLM calls itself.** It calls `judge.llm_fn()` — a user-provided function.
4. **Cost estimate shown before any judge calls are made.** User must confirm or pass `confirm=False`.
5. **No composite metric (no JCI).** Report shows individual bias scores and ranks only.
6. **Pre-shipped fixtures only.** No helper LLM in v0.1.
7. **Stateless deployment assumed.** Deployment-dependent biases skipped with clear note.
8. **Five domains.** Q&A, summarization, code, safety, dialogue.
9. **Three judge types.** Pointwise, pairwise, binary.
10. **Seven bias tests.** Position, verbosity, self-consistency, scale anchoring, authority, concreteness, yes-bias.
11. **`domain` is required** on `JudgeAuditor` — no implicit default.
12. **No fixture-loader fallback magic.** Each bias test declares its fixture path explicitly.
13. **Stdlib dataclasses only.** No pydantic — validation via lightweight `__post_init__`.

## 4. Public API — Full Specification

### Judge dataclass

```python
from dataclasses import dataclass, field
from typing import Callable

VALID_JUDGE_TYPES = {"pointwise", "pairwise", "binary", None}

@dataclass
class Judge:
    llm_fn: Callable[[str], str]   # required: wraps user's LLM call
    system_prompt: str             # required: judge's persona/role
    eval_template: str             # required: template with {placeholders}

    # Optional metadata (for report labeling only)
    judge_name: str = "unnamed_judge"
    model_name: str = "unknown"

    # Optional override (skip auto-detection)
    judge_type: str | None = None  # "pointwise" | "pairwise" | "binary" | None

    def __post_init__(self) -> None:
        if self.judge_type not in VALID_JUDGE_TYPES:
            raise ValueError(
                f"judge_type must be one of {VALID_JUDGE_TYPES}, got {self.judge_type!r}"
            )
        if "{" not in self.eval_template or "}" not in self.eval_template:
            raise ValueError(
                "eval_template must contain at least one {placeholder}"
            )
```

### JudgeAuditor

```python
class JudgeAuditor:

    ): ...

```

### AuditReport

```python
@dataclass
class AuditReport:
    # Judge info
    judge_name: str
    judge_type: str        # lowercase: "pointwise" | "pairwise" | "binary"
    domain: str
    timestamp: str

    # Per-test results (no composite)
    tests: dict[str, BiasResult]

    # Convenience (all exclude not_applicable results)
    def failed_tests(self) -> list[BiasResult]: ...    # verdict == "FAIL"
    def passed_tests(self) -> list[BiasResult]: ...    # verdict == "PASS"
    def ranked(self) -> list[BiasResult]: ...          # worst bias first (applicable only)
    def summary(self) -> str: ...                      # console-ready text

    # Export
    def save_json(self, path: str) -> None: ...
    def save_html(self, path: str) -> None: ...
```

### BiasResult

```python
@dataclass
class BiasResult:
    test_name: str
    score: float                  # 0..1, higher = more calibrated (0.0 if not_applicable)
    verdict: str                  # "PASS" | "FAIL" | "N/A"
    n_fixtures: int               # how many items the test consumed (0 if not_applicable)
    details: dict = field(default_factory=dict)   # raw measurements (test-specific)
    examples: list[dict] = field(default_factory=list)  # 2-3 concrete failure examples
    severity: str | None = None   # "CRITICAL"|"SIGNIFICANT"|"MINOR"|"NONE"|None (for N/A)
    rank: int = 0                 # 1 = worst bias among applicable tests; 0 = N/A or unranked
    not_applicable: bool = False  # True if this test skipped for this judge type
    skip_reason: str = ""         # e.g. "position bias only applies to pairwise judges"
```

**Severity ↔ verdict overlap (by design):**
- score < 0.50 → CRITICAL (always FAIL)
- 0.50 ≤ score < 0.65 → SIGNIFICANT (always FAIL)
- 0.65 ≤ score < 0.80 → MINOR (FAIL if score < 0.70, else PASS)
- score ≥ 0.80 → NONE (always PASS)

The MINOR band straddles the 0.70 PASS line — a "PASS but MINOR" result is intentional
("technically passing, watch for drift"). Documented in `docs/methodology.md`.

## 5. Cost Estimation

The library knows call counts exactly (deterministic, based on fixture counts).
The library does NOT know cost per call (model is inside the user's black-box callable).

### Design

```python
@dataclass
class CostEstimate:
    total_calls: int
    calls_per_test: dict[str, int]
    estimated_cost_usd: float | None   # None if cost_per_call not provided
    cost_per_call: float | None

    def display(self) -> None:
        print(f"\nEstimated judge calls: {self.total_calls}")
        print(f"\nBreakdown by test:")
        for test, calls in self.calls_per_test.items():
            print(f"  {test:<20} {calls} calls")

        if self.estimated_cost_usd is not None:
            print(f"\nEstimated cost: ${self.estimated_cost_usd:.2f} USD")
            print(f"(based on ${self.cost_per_call:.4f} per call — rough estimate;")
            print(f" actual cost depends on prompt + response length)")
        else:
            print(f"\nTip: pass cost_per_call=<USD per call> to see cost estimate.")
            print(f"  OpenAI gpt-4o:        ~$0.005 per call")
            print(f"  OpenAI gpt-4o-mini:   ~$0.0003 per call")
            print(f"  Anthropic claude-3.5: ~$0.006 per call")
```

### Call count calculation

```python
CALLS_PER_TEST = {
    "position":         lambda n: n * 2,   # each pair scored in both A/B orders
    "verbosity":        lambda n: n * 2,   # original + padded
    "self_consistency": lambda n: n * 5,   # same item × 5 runs
    "scale_anchoring":  lambda n: n,       # one call per tiered example
    "authority":        lambda n: n * 2,   # original + authority-framed
    "concreteness":     lambda n: n * 2,   # vague + concrete
    "yes_bias":         lambda n: n,       # one call per T/F probe
}
```

### Confirmation prompt (when confirm=True)

```
Judicator Audit — Cost Estimate
────────────────────────────────────────
Judge:   my_dialogue_judge
Domain:  dialogue
Type:    pointwise [auto-detected]

Estimated judge calls: 580

  verbosity          100 calls
  self_consistency   200 calls  (40 items × 5 runs)
  scale_anchoring    200 calls
  authority          100 calls
  concreteness        28 calls  (14 items × 2 variants)
  [position: N/A for pointwise]
  [yes_bias: N/A for pointwise]

Tip: pass cost_per_call=0.005 to see USD estimate.

Proceed? [Y/n]:
```

If user types `n` or `N` → raise `AuditCancelled` exception, no judge calls made.
If `confirm=False` → skip prompt, run immediately.
If `confirm=True` but stdin is **not a TTY** (CI / piped input) → log a warning and
skip the prompt automatically (no hanging on `input()` in non-interactive contexts).

**Detection logic:**
```python
def detect_judge_type(eval_template: str) -> tuple[JudgeType, float]:
    """Returns (JudgeType, confidence). Confidence in [0, 1]."""
    placeholders = set(re.findall(r'\{(\w+)\}', eval_template))

    if {'response_a', 'response_b'}.issubset(placeholders):
        return JudgeType.PAIRWISE, 0.95

    binary_keywords = ['yes or no', 'true or false', 'answer only yes',
                       'answer only no', 'respond with yes', 'respond with no']
    if any(kw in eval_template.lower() for kw in binary_keywords):
        return JudgeType.BINARY, 0.90

    response_slots = {'response', 'answer', 'output', 'completion'}
    if response_slots & placeholders:
        return JudgeType.POINTWISE, 0.85

    return JudgeType.UNKNOWN, 0.0
```

If confidence < 0.8 and user did not pass `judge_type` explicitly:
raise `DetectionError` with message asking user to pass `judge_type="pointwise"` etc.

**Parser contracts (`parsers.py`):**
```python
def parse_pointwise(text: str) -> float | None:
    """Extract first number in [1, 10] from judge output. None if not found."""

def parse_pairwise(text: str) -> str | None:
    """Return 'A' or 'B' (first verdict token found). None if neither."""

def parse_binary(text: str) -> str | None:
    """Return 'Yes' or 'No'. None if neither."""
```
On parse failure (`None`): the bias test treats that single call as a no-op (logged, not counted as bias). If >25% of a test's calls fail to parse, the test returns `verdict="FAIL"`, `severity="CRITICAL"`, `details={"parse_failure_rate": ...}` — judge output is unusable for that test.

```python
def load_fixtures(rel_path: str, max_items: int | None = None) -> list[dict]:
    """
    Load fixtures from a relative path under src/judicator/fixtures/data/.

    Examples:
      load_fixtures("qa/verbosity.jsonl")
      load_fixtures("universal/concreteness.jsonl")
      load_fixtures("qa/position.jsonl", max_items=20)

    Raises FixtureNotFound if file does not exist.
    No domain-fallback magic — each bias test declares its exact fixture path.
    """
    path = DATA_DIR / rel_path
    if not path.exists():
        raise FixtureNotFound(f"No fixtures at {rel_path}")

    items = [json.loads(line) for line in path.read_text(encoding="utf-8").splitlines() if line.strip()]
    if max_items:
        items = items[:max_items]
    return items
```

**BiasTest interface:**
```python
class BiasTest(ABC):
    name: str
    applicable_types: list[JudgeType]
    fixture_path: str           # e.g. "{domain}/verbosity.jsonl" or "universal/yesno_probes.jsonl"

    @abstractmethod
    def run(
        self,
        judge: Judge,
        fixtures: list[dict],
        call_counter: CallCounter
    ) -> BiasResult:
        ...

    def is_applicable(self, judge_type: JudgeType) -> bool:
        return judge_type in self.applicable_types
```

Note: **no `weight` field.** With no composite metric, weight had no purpose.

**Position bias test (uses winner_response / loser_response from JudgeBench):**
```python
class PositionBiasTest(BiasTest):
    name = "position"
    applicable_types = [JudgeType.PAIRWISE]
    fixture_path = "{domain}/position.jsonl"

    def run(self, judge, fixtures, call_counter) -> BiasResult:
        # Positional consistency: a calibrated judge gives the SAME content-verdict
        # regardless of A/B ordering. Inconsistency = position bias.
        inconsistent = 0
        slot_a_picks = 0  # secondary stat: directional skew (favors slot A)
        usable = 0

        for item in fixtures:
            # Run 1 — winner in slot A → calibrated answer is "A"
            v_AB = parse_pairwise(judge.llm_fn(judge.eval_template.format(
                question=item["question"],
                response_a=item["winner_response"],
                response_b=item["loser_response"])))
            call_counter.increment()
            # Run 2 — winner in slot B → calibrated answer is "B"
            v_BA = parse_pairwise(judge.llm_fn(judge.eval_template.format(
                question=item["question"],
                response_a=item["loser_response"],
                response_b=item["winner_response"])))
            call_counter.increment()

            if v_AB is None or v_BA is None:
                continue
            usable += 1
            # Inconsistent = judge flipped its preference based on slot, not content.
            # Calibrated: AB→"A" and BA→"B" (winner picked both times).
            if v_AB == v_BA:
                inconsistent += 1
            if v_AB == "A": slot_a_picks += 1
            if v_BA == "A": slot_a_picks += 1

        inconsistency_rate = inconsistent / usable if usable else 0.0
        score = round(1 - inconsistency_rate, 3)

        return BiasResult(
            test_name="position",
            score=score,
            verdict="PASS" if score >= 0.7 else "FAIL",
            n_fixtures=usable,
            details={
                "inconsistency_rate": inconsistency_rate,
                "slot_a_pick_rate":   slot_a_picks / (2 * usable) if usable else 0.0,
            },
            examples=extract_failures(...),
            severity=severity_from_score(score),
        )
```

**Verbosity bias test (uses original_response / manipulated_response):**
- Score both. Measure mean inflation: `mean(padded_score - original_score)`.
- Score = `1 - clip(mean_inflation / scale_range * 2, 0, 1)`.

**Self_consistency test:**
- For each of 40 items: call judge 5×, record scores.
- Compute per-item variance, aggregate to mean variance.
- Normalize to [0, 1]: `score = 1 - mean_variance / max_possible_variance`.

**Success criteria:**
- Position test on `make_biased_judge("position")` → FAIL, score < 0.3
- Verbosity test on `make_biased_judge("verbosity")` → FAIL, score < 0.5
- Self_consistency test on deterministic mock judge → PASS, score = 1.0
- All tests on `make_biased_judge("none")` → PASS

**Scale anchoring (2-bucket):**
```python
class ScaleAnchoringTest(BiasTest):
    name = "scale_anchoring"
    applicable_types = [JudgeType.POINTWISE]
    fixture_path = "universal/scale_tiers.jsonl"

    def run(self, judge, fixtures, call_counter) -> BiasResult:
        high_scores, low_scores = [], []
        for item in fixtures:
            score = parse_pointwise(judge.llm_fn(format(item)))
            call_counter.increment()
            (high_scores if item["tier"] == "high" else low_scores).append(score)

        # Test math uses only the extremes — middle tiers don't exist.
        spread = mean(high_scores) - mean(low_scores)
        scale_range = 9.0  # assume 1-10 scale; documented limitation
        expected_spread = scale_range * 0.7
        score = round(min(1.0, spread / expected_spread), 3)
        ...
```

**Authority test:** Same structure as verbosity — `original_response` vs `manipulated_response` pairs from `{domain}/authority.jsonl`. Score both; measure mean inflation.

**Concreteness test:** Uses **natural pairs** from OffsetBias `concreteness` subcategory.
Each item has `vague_response` (preferred / correct) and `concrete_response` (specifics-loaded / biased).
The data is naturally paired (not author-manipulated) — the test asks "does the judge inflate on fabricated specifics?"

Per judge type:
- **Pointwise:** score both responses. Calibrated = `vague_score >= concrete_score`. Final score = fraction of items where judge was calibrated.
- **Pairwise:** present (vague_response, concrete_response) as (A, B) and as (B, A). Calibrated = vague picked in both orderings.
- **Binary:** N/A. The fixture is small (n=14) and the rate-delta metric a binary judge would need is too noisy to be meaningful. `applicable_types = [JudgeType.POINTWISE, JudgeType.PAIRWISE]`.

**Yes-bias:**
```python
# yesno_probes.jsonl: balanced T/F statements (ground_truth ∈ {"Yes", "No"})
# Score each (judge.llm_fn → parse_binary → "Yes" | "No" | None)
# Use balanced accuracy so a "always-No" judge doesn't score 1.0.
correct = sum(parse_binary(judge.llm_fn(...)) == item["ground_truth"] for item in fixtures)
score = round(correct / n_usable, 3)  # n_usable = items where parse succeeded
# Report fp_rate (saying Yes to No-statements) and fn_rate separately in details.
```

```python
class JudgeAuditor:
    def audit(self) -> AuditReport:
        # 1. Detect judge type
        judge_type, confidence = detect_judge_type(self.judge.eval_template)
        if self.judge.judge_type:
            judge_type = JudgeType(self.judge.judge_type)  # user override

        # 2. Probe call: verify template fills and output parses
        self._probe(judge_type)

        # 3. Select applicable tests
        applicable = [t for t in ALL_TESTS if t.is_applicable(judge_type)]
        if self.tests:
            applicable = [t for t in applicable if t.name in self.tests]

        # 4. Load fixtures (skip tests with missing fixture files)
        fixture_map = {}
        skipped = []
        for test in applicable:
            try:
                path = test.fixture_path.format(domain=self.domain)
                fixture_map[test.name] = load_fixtures(
                    path, max_items=self.max_items_per_test,
                )
            except FixtureNotFound:
                skipped.append((test, f"no fixtures at {path}"))

        runnable = [t for t in applicable if t.name in fixture_map]

        # 5. Estimate + confirm
        estimate = self._estimate(runnable, fixture_map)
        estimate.display()
        if self.confirm:
            self._ask_confirmation()  # auto-skips on non-TTY with warning

        # 6. Run tests
        call_counter = CallCounter()
        results = {}
        for test in runnable:
            results[test.name] = test.run(
                self.judge, fixture_map[test.name], call_counter,
            )

        # 7. Add N/A entries for non-applicable tests + missing-fixture skips
        skip_reasons = {t.name: r for t, r in skipped}
        for test in ALL_TESTS:
            if test.name in results:
                continue
            reason = skip_reasons.get(
                test.name,
                f"{test.name} does not apply to {judge_type.value} judges",
            )
            results[test.name] = BiasResult(
                test_name=test.name,
                score=0.0,
                verdict="N/A",
                n_fixtures=0,
                not_applicable=True,
                skip_reason=reason,
            )

        # 8. Rank applicable results (worst first by score ascending)
        ranked = sorted(
            [r for r in results.values() if not r.not_applicable],
            key=lambda r: r.score,
        )
        for i, r in enumerate(ranked):
            r.rank = i + 1

        # 9. Build and return report
        return AuditReport(
            judge_name=self.judge.judge_name,
            judge_type=judge_type.value,
            domain=self.domain,
            timestamp=utcnow(),
            tests=results,
        )
```

**Confirmation prompt (TTY-aware):**
```python
def _ask_confirmation(self) -> None:
    if not sys.stdin.isatty():
        warnings.warn("Non-TTY detected; skipping confirm prompt.")
        return
    answer = input("Proceed? [Y/n]: ").strip().lower()
    if answer == "n":
        raise AuditCancelled("user declined to proceed")
```

    if score < 0.50:  return "CRITICAL"      # always FAIL
